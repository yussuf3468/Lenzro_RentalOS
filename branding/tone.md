# Lenzro — Voice & Tone

Lenzro writes the way it engineers: clear, precise, confident, never inflated.

## Principles

1. **Plain over clever.** Say the thing. No buzzwords, no filler.
2. **Confident, not boastful.** State capability and evidence; let the work speak.
3. **Respect the reader's time.** Lead with the outcome. Short sentences. Scannable structure.
4. **Precise.** Concrete numbers, dates, deliverables, and owners — never vague.
5. **Warm professionalism.** Courteous and human, never stiff or corporate-robotic.

## Document conventions

- Dates: `27 June 2026` (long form) in prose; `2026-06-27` (ISO) in metadata/IDs.
- Currency: explicit code + symbol, e.g. `KES 1,250,000` or `USD 4,500`.
- Numbers: thousands separators; tabular figures in tables.
- Reference IDs: `LNZ-<TYPE>-<YYYY>-<SEQ>` e.g. `LNZ-PROP-2026-0042`.
- Headings: sentence case for body sections; title case for document titles.
- Voice: active. "We will deliver…" not "It is anticipated that delivery…".

## Words we prefer / avoid

- Prefer: deliver, scope, milestone, outcome, ownership, timeline, investment.
- Avoid: synergy, leverage (as verb), world-class, cutting-edge, revolutionary, ninja, rockstar.
